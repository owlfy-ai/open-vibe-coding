# Code of Conduct

## Our Pledge

We are committed to making participation in this project a respectful and welcoming experience for everyone.

## Expected Behavior

- Be kind and constructive.
- Respect different experience levels and backgrounds.
- Keep discussions focused on improving the project.
- Give and receive feedback with patience.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Deliberately disruptive behavior in issues, discussions, or pull requests.

## Enforcement

Project maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior violates this code of conduct.

If you need to report a conduct issue, open a GitHub issue with a clear description. Do not include private personal information publicly.
