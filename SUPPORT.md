# Support

Open Vibe Coding is provided as an open source project.

## Getting Help

Use GitHub issues for:

- bug reports
- documentation problems
- feature requests
- build or setup questions

Before opening an issue, please check the existing issues and documentation.

## Security Reports

For security-related reports, see [SECURITY.md](./SECURITY.md). Do not post secrets, tokens, API keys, private prompts, or exploit details publicly.

## Maintainer Availability

Response times are not guaranteed. Please provide clear reproduction steps and environment details to make issues easier to investigate.
